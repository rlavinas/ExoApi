# ExoApiProjeto de API para controle de projetos da empresa ExoApi.## Versão: 1.0## Requistos necessários.1) Realizar instalação Visual Studio 2021. (.NET 6.0)   - Componentes instalados:     - Microsoft.AspNetCore.Authentication.JwtBearer - 6.0.12     - Microsoft.EntityFrameworkCore.SqlServer - 7.0.1     - Swashbuckle.AspNetCore.Swagger - 6.4.0     - Swashbuckle.AspNetCore.SwaggerGen - 6.4.0     - Swashbuckle.AspNetCore.SwaggerUI - 6.4.03) Realizar instalação SQL Server Express.4) Realizar instalação Insomnia (ultima versão).## Procedimentos instalação1) Clonar repositório pelo link: https://github.com/rlavinas/ExoApi.git2) Executar o script para criação da base de dados "ExoApi" conforme arquivo "docs/DDL Script ExoApi.sql"3) Abrir o projeto ExpApi => ExoApi.sln.## Procedimentos de utilização inicial1) Para utilização é necessário realizar a criação de um login pelo endpoit api/Usuarios [POST]2) Realizar o login com o usuário (email) e senha criados. Vai retornar o token Jwt. Cada token tem 30 minutos de utilização antes da expiração.3) Após a criação e posterior autenticação do usuário, todos os demais serviços estarão disponíveis, pois dependem de autenticação.## Lista de serviços disponíveis1) Acessar o Swagger em http://localhost:5265/swagger e verificar a lista.   - Usuario.   - Login.   - Projeto.   - Atividade.
